- [ ] BSM / BSD (Blind Spot Monitoring - เตือนจุดอับสายตา)

- [ ] ACC (Adaptive Cruise Control - ควบคุมความเร็วอัตโนมัติ)
- [ ] ICA (Intelligent Cruise Assist - ระบบช่วยขับอัจฉริยะ)
- [ ] Traffic Jam Assist (ช่วยขับในการจราจรติดขัด)
- [ ] Auto High Beam (ปรับไฟสูงอัตโนมัติ)
- [ ] Night Vision (มองเห็นในความมืด)
- [ ] Driver Monitoring System (ตรวจจับความเหนื่อยล้าคนขับ)
- [ ] Speed Limit Recognition (อ่านป้ายจำกัดความเร็ว)
- [ ] Traffic Sign Recognition (อ่านป้ายจราจร)
